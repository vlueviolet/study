JavaScript vs. JQuery

  • 자바스크립트는 모든 브라우저 대응이 안되지만, jquery는 가능하다.
  • 특히 선택자 사용이 편리하다는게 가장 큰 장점

자바스크립트의 요소 선택


	document.getElementById(‘container’);
	document.getElementsByClass(‘box);	//IE8이상 지원 안됨

jquery의 요소 선택


	$(‘.box’), $(‘#container’);

이벤트와 이벤트에 의한 행동

이벤트를 주는 방법


	$('.box').on('click', function(){

	});

함수 선언

선언함수


	changeCss(); 	//함수선언 이전에 호출해도 에러가 나지 않는다.
	function changeCss(){
		$('.box').css('border','1px solid red');
	}

익명함수

호출하는 위치의 혼란을 줄이기위해 익명함수로 함수를 선언한다. (회사 컨벤션)


	var changeCss = function(){
		$(‘.box’).css();
	}
	changeCss();	//함수호출이 선언보다 위에 있다면 에러남

jquery 시작

  • dom태그를 모두 읽고 시작해라란 의미, 라이브러리 읽는 성능때문에 하단에 두긴함
  • head에 두더라도 얘는 읽고 저장해놓는거지 돔읽고 다시 위로 올라와서 해석하는건 아님
	$(function(){});
	또는
	$(document).ready{}

jqueyr의 사용

jquery나 자바스크립트를 사용하면 성능이슈가 있기때문에 마크업에서 표현할 수 있는 것들은 다하고, jquery에서는 겸손하게만 쓰자.
